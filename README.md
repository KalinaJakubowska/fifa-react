# Fifa match generator
## [DEMO](https://kalinajakubowska.github.io/fifa-react/)

Match generating tool can be used to create matches list for tournamenst. It's possible to generate 1vs1 and 2vs2 matches queue.
## Example usage
Start with completing players list.

![Example players list](https://github.com/KalinaJakubowska/fifa-react/blob/master/images/examplePlayersList.png?raw=true)

Than choose single or Volta (2vs2) version. Application will generate all matches. Enter the results in the appropriate places, application is going to show all the points automatically.

![Example usage](https://github.com/KalinaJakubowska/fifa-react/blob/master/images/exampleUsage.gif?raw=true)
